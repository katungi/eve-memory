# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish packages.

## What is a changeset?

A changeset is a piece of information about changes made in a branch or commit. It holds three bits of information:

1. What packages should be published
2. What version bump type (major, minor, patch) each package should receive
3. A changelog entry for each package

## When should I add a changeset?

You should add a changeset when you make changes that require a version bump and changelog entry for `eve-memory`.

Private packages (`@eve-memory/cli`, `@eve-memory/adapter-pg`) are ignored by Changesets.
